В 3 задании get запросы обрабатываются на localhost:3000

1) api/document -- ищет поставки по дате договора
	параметры:
		year=YYYY
			ищет поставки за определенный год
		quarter=[1-4]
			ищет поставки за определенный квартал
		month=MM
			ищет поставки за определенный месяц
		day=dd
			ищет поставки за определенный день
		authorized=false/true
			ищет только авторизованные поставки, если значение true

2) api/posting -- ищет поставки по дате поставки
	параметры:
	year=YYYY
		ищет поставки за определенный год
	quarter=[1-4]
		ищет поставки за определенный квартал
	month=MM
		ищет поставки за определенный месяц
	day=dd
		ищет поставки за определенный день
	authorized=false/true
		ищет только авторизованные поставки, если значение true
